Eastleigh Balance Dashboard V2

Files:
- app.py
- Eastleigh_Masterlist.xlsx
- requirements.txt

Run locally:
1. source venv/bin/activate
2. pip install -r requirements.txt
3. python3 app.py
4. open http://127.0.0.1:8050

For Render:
Build command:
pip install -r requirements.txt

Start command:
python app.py
